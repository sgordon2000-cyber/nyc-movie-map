# NYC Movie Map

A interactive map visualization of my cinematic journey through New York City's diverse theater landscape. This project tracks every movie I've seen in NYC theaters since 2023, complete with ratings, theater details, and social connections.

## 🎬 Overview

"Seth's Cinephile City" is a personal diary of movies by theaters, documenting my film-watching adventures across NYC's rich cinema scene. Each theater marker represents a unique venue where I've experienced cinema, from historic arthouse theaters to modern multiplexes, all connected by the films and friends that brought me there.

## ✨ Features

### Interactive Map
- **Color-coded theaters** based on average ratings:
  - 🟢 Green: Excellent (4.0+ average rating)
  - 🟠 Orange: Good (2.5-3.9 average rating)
  - 🔴 Red: Below average (< 2.5 average rating)
  - ⚪ Gray: No ratings recorded

### Theater Information
- **Movie counts**: Total films seen at each venue
- **Rating statistics**: Average rating, best/worst films
- **Recent activity**: Most recently watched film
- **Letterboxd integration**: Direct links to theater-specific film lists

### Social Filtering
Filter theaters by the people I've watched movies with:
- Jen, Seneca, WJ, Avery, Charlotte, Jackson, Adi, Julia, Emma, Caroline, Adam, Ben, Asja, Bridget, Livi, Adrian

## 🚀 How to Use

1. **View the Map**: Open `index.html` in any modern web browser
2. **Explore Theaters**: Click on any theater marker to see details
3. **Filter by People**: Use the filter buttons to show only theaters visited with specific friends
4. **Access Letterboxd**: Click theater names to view all films from that venue on Letterboxd

## 📊 Data

The map is built from personal movie diary data including:
- Theater names and locations (latitude/longitude)
- Movie titles and watch dates
- Personal ratings (1-5 scale)
- Social tags (who joined me)
- Letterboxd links for each film
